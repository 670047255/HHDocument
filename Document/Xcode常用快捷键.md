# Xcode常用快捷键

### Xcode导航快捷键


* Command + 1： 快速浏览代码、图片以及用户界面文件

* Command + 0： 打开/关闭左边导航栏： 

* Command + option + 0： 关闭/打开右边工具栏：

### Xcode搜索快捷键

* Command + Shift + F：搜索导航器

* Command + F: 文件内搜索

* Command + G: 搜索下一处

* Shift + Command + G: 搜索上一处

* 快速打开：Command + Shift + O

###Xcode快捷工具


* Command + Shift + 0 (Zero) ：查看苹果文档和参考

* Control + Command +  ↑ / ↓ ：程序中(OC) .h 和 .m文件间的快速切换

* Command + Shift + K：清除工程

* Command + Shift + J ：展示当前你在工程导航器中打开的文件

* Command + J：展示小视图框，切换焦点 （配合上面的使用）

* Command + Option + Shift + ←：收缩方法

* Command + Option + Shift + →：展开方法 

* Control + Command + ← ： 回到 之前打开的页面

* Control + Command + → ： 回到 之前打开的页面

* Control + Command + ↑ ： 跳到页面最上面

* Control + Command + ↓ ：跳到页面最下面



### Xcode编辑代码快捷键



* Command + [ ：左缩进

* Command + ] ：右缩进

* Control-F： 向右一个字符（forward）

* Control-B：向左一个字符（backward）

* Control-P： 前一行（previous）

* Control-N： 后一行（next）

* Control-A： 去行首

* Control-E： 到行尾（end）

* Control-T： 调换光标两边的字符（transpose）

* Control-D： 删除光标右侧字符（delete）

* Control-K： 删除本行剩余的字符（kill）
